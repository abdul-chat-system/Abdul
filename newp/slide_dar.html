2. Automatic Password Creation After Signup

One of the standout features of Abdul Chat System is that the password is automatically generated upon successful registration.

This ensures security and simplicity for users, as they don’t need to think of a password themselves.

The system generates a random yet memorable password using a mix of letters and numbers (e.g., Q3T5 or X7P9). The generated password is sent via email or displayed on-screen after signup, with an option for the user to change it later.

This feature improves security by reducing the risk of weak passwords and enhances user experience by simplifying the signup process.

6. Things That Can Be Shared in Abdul Chat System

The Abdul Chat System supports multiple types of media sharing to enhance user experience and make conversations more interactive. Below is a detailed explanation of each type of content that users can share:

  • i. SMS (Text Messages).

Users can send and receive instant text messages in real time, Messages appear in chat bubbles with timestamps, Typing indicators show when the other person is typing.
Users can edit or delete messages within a given time frame, Supports emoji reactions for quick responses.

  • ii. Photos (Image Sharing).

Users can send and receive images within the chat. The system supports common image formats: JPEG, PNG, GIF, and WEBP.
Photos appear as thumbnails, and users can click to enlarge. There is an option to download images directly.
Supports image captions to describe the photo.

  • iii. Videos (Video Sharing).

Users can upload and send videos directly from their device. Supported formats: MP4, AVI, and MOV.
Videos are displayed with a play button, allowing in-app playback. Users can choose to watch videos in full screen.
Video size limit can be adjusted in system settings to manage server storage.

  • iv. Audio (Voice Messages & Audio Files).

Users can record and send voice notes directly from the chat. Supports sending pre-recorded audio files like music or podcasts. Common audio formats: MP3, WAV, AAC.
Users can play audio messages without downloading. Audio messages show a progress bar for playback control.

  • v. Documents (File Sharing).

Users can share documents and files for work or personal use.

Supported document types:

✅ PDF (eBooks, reports, invoices)
✅ Word (DOCX) (text files, resumes)
✅ Excel (XLSX) (spreadsheets, financial reports)
✅ PowerPoint (PPTX) (presentations)
✅ ZIP/RAR (compressed files)
Users can preview certain files (like PDFs) before downloading.

The Abdul Chat System offers two modes to enhance user experience, especially in different lighting conditions:

  • i. Light Mode.

Default setting: A bright interface with a clean, light background (usually white or light gray). The text is dark (usually black or dark gray) for easy readability.
The buttons and icons are designed with vibrant colors to create a fresh, clean look. Ideal for use during the day or in well-lit environments.

  • ii. Night Mode.

Dark interface with a dark background (usually dark gray or black) to reduce eye strain in low-light settings.
The text is light-colored (white or light gray) to ensure contrast and readability.
Buttons and icons are often toned down, typically using muted or blue accents, to complement the dark background.
Ideal for use at night or in dimly lit environments to reduce blue light exposure.
